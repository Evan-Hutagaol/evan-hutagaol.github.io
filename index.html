Projects Here are some of the projects I had the opportunity to work on during my time at Institut Teknologi Del

Sistem Informasi Tepi Danau Bistro

Tepi Danau Bistro is a charming cafe situated on the edge of Lake Toba, located in Lumban Silintong, Kabupaten Toba, North Sumatra. I developed an innovative and engaging web application for the bistro, which serves as a comprehensive resource for visitors and potential customers. The platform showcases facility details, menu prices, images, and management information. Additionally, it features an integrated Google Maps tool to help visitors easily find the cafe's location.
Tech stack: HTML & CSS, Bootstrap, PHP Native, JavaScript, MySQL

GitHub Repository

Sistem Informasi Fasilitas IT DEL

Developed a comprehensive web-based information system for Institut Teknologi Del, a private university located in Laguboti, Kabupaten Toba, North Sumatra, founded by Mr. Luhut Pandjaitan. This application is designed to provide detailed information about the university's facilities, including a floor plan to assist visitors in navigating the campus. It also features updates on facilities under construction and includes a suggestion tool for collecting visitor feedback. This system enhances the visitor experience by offering valuable insights and facilitating easier access to campus information.
Tech stack: HTML & CSS, Bootstrap 5, Laravel, JavaScript, MySQL

GitHub Repository

Sistem Informasi BPBD Kabupaten Toba

Developed an innovative and practical application for BPBD Toba, a dedicated agency for disaster management in Toba District. This app is designed to provide critical information to the public during disasters, offer educational resources on minimizing disaster impact, and include a reporting feature for users to report local incidents. The system also integrates weather forecasts, the latest news, and disaster mitigation strategies to keep the community informed and prepared.
Tech stack: HTML & CSS, Bootstrap 5, Laravel, JavaScript, MySQL
